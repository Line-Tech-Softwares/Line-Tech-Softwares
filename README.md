## 🔌 API & SDKs

Line Tech Softwares provides a comprehensive API and official SDKs for developers to integrate with our services.

### API Server
- **Location**: `backend/` directory
- **Technology**: Node.js + Express
- **Features**: RESTful API with authentication, rate limiting, and CORS support
- **Endpoints**: User management, project management, authentication

### JavaScript SDK
- **Location**: `sdk-js/` directory
- **Installation**: `npm install linetec h-sdk`
- **Features**: Complete API client with error handling and TypeScript support

### Python SDK
- **Location**: `sdk-python/` directory
- **Installation**: `pip install linetec h-sdk`
- **Features**: Full API client with Pydantic models and async support

For detailed API documentation and SDK usage, see [Developer Docs](developer%20docs.md).

## 🚀 Getting Started

### Prerequisites
- Modern web browser
- Text editor (VS Code recommended)
- Local server (optional, for development)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/linetechsoftwares/website.git
   cd website
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - For development, use a local server:
     ```bash
     # Using Python
     python -m http.server 8000

     # Using Node.js
     npx serve .

     # Using PHP
     php -S localhost:8000
     ```

3. **Navigate the site**
   - Visit `http://localhost:8000` (or your server URL)
   - Explore different sections and tools
